SOBRE

Tag Filter é uma aplicação para acompanhamento de tags no Twitter.
Construido em Python, Tag Filter foi desenvolvido como parte do processo seletivo da empresa Magrathea Labs.

Para construção da solução foi utilizado o wrapper Tweepy. Uma vez construído o handler das mensagens filtradas, foi utilizado Flask para criação de uma interface web amigável.
